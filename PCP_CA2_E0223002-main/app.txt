Dataset B - Fitness Tracker Activities 
Each activity object obtains :
1. activityId
2. name
3. steps
4. caloriesBurned 
5. workoutMinutes
6. goalAcheived
7. date

Dataset contains inconsistent and invalid entires. students must handle them properly 
Application requirements 
U must build react applicaation using 
1. react context
2. useReducer
3. React router


Mandatory routes
/activities
/activities/:id
/filter
/stats

2. All updates must go thryugh Reducer only 
3. No hardcoding

2. caloriesBurned >0
